# Tetris / 终端命令行俄罗斯方块

基于 tico 引擎开发的经典俄罗斯方块终端游戏。

## 游戏特色

### 经典俄罗斯方块机制
- **7-Bag 随机算法** - 确保每7个方块为一组完整出现，避免长时间不出某个方块
- **幽灵方块** - 显示当前方块将要落下的位置
- **硬降** - 按空格键让方块瞬间落到底部
- **软降** - 按下方向键加速下落
- **预览功能** - 提前看到下一个方块

### 7种方块

| 方块 | 颜色 | 描述 |
|------|------|------|
| I | 青色 | 长条方块，一次消4行的最佳选择 |
| O | 黄色 | 正方形，最稳定的方块 |
| T | 洋红 | T形方块，适合做T-Spin技巧 |
| S | 绿色 | S形，与Z形互为镜像 |
| Z | 红色 | Z形，经典S形的搭档 |
| J | 蓝色 | J形，左撇子版的L形 |
| L | 黄色 | L形，右撇子版的J形 |

### 计分系统

| 消除行数 | 得分 (× 等级) |
|----------|---------------|
| 1行      | 100           |
| 2行      | 300           |
| 3行      | 500           |
| 4行      | 800 (Tetris!) |

每消除10行升一级，下落速度会逐渐加快。

### 操作说明

| 按键 | 功能 |
|------|------|
| ← → 或 A/D | 左右移动 |
| ↑ 或 W | 旋转方块 |
| ↓ 或 S | 软降（加速下落） |
| 空格 | 硬降（瞬间落底） |
| P / Esc | 暂停游戏 |
| Q / Esc | 返回菜单 |

### 游戏模式

- **开始游戏** - 使用7-Bag随机算法开始新游戏
- **游戏记录** - 查看前10名最高分（本地保存）
- **退出游戏** - 退出程序

### 特色功能

- **本地存档** - 游戏记录保存在 `records.json` 文件中
- **中文界面** - 完整的中文用户界面
- **暂停/继续** - 随时可暂停游戏
- **实时统计** - 实时显示分数、行数、等级和下落速度

## 如何运行

```bash
npm run example:tetris
```

或直接运行：

```bash
node src/index.js
```

## 技术栈

- **引擎**: [tico](https://github.com/omegod/tico) - 终端游戏引擎
- **运行环境**: Node.js
- **渲染**: ASCII 艺术 + ANSI 颜色
- **资源**: JSON 格式的方块定义
